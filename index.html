Bienvenue dans notre outils de fruit tracker

Avant de pouvoir utiliser cet outil vous devez compléter les fonctions getRandomElement et track dans le fichier index.ts. La fonction doit :

  • Prendre 1 paramètre qui sera un tableau
  • Vérifier que ce tableau contienne au moins 5 éléments
  • Retourner true si oui, sinon false
